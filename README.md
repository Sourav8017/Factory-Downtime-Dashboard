# 📊 Factory Downtime Analysis Dashboard

## 🚀 Project Overview

This project analyzes industrial telemetry data to identify downtime trends, high-risk machines, and operational inefficiencies using Tableau.

---

## 🎯 Objectives

* Monitor total system downtime
* Identify machines contributing most to failures
* Analyze downtime trends over time
* Compare performance across factories and device types

---

## 🛠️ Tools Used

* Tableau (Data Visualization)
* JSON Dataset (Industrial Telemetry Data)

---

## 📊 Dashboard Features

* **KPI**: Total Downtime
* **Downtime by Factory**
* **Downtime by Device Type**
* **Downtime Trend Over Time**
* **Top 10 Failing Machines**
* Interactive filters for dynamic analysis

## 📁 Dataset

The dataset is included as a compressed ZIP file due to size limitations.

### 🔽 How to Use Dataset:

1. Download the ZIP file from this repository
2. Extract it
3. Place the `.json` file inside the `data/` folder
4. Open the Tableau workbook (`.twb`)

---

## 🧠 Key Insights

* Laser machines show the highest downtime
* A small number of devices contribute to most failures
* Downtime occurs in spikes, indicating intermittent issues
* Certain factories have higher failure rates
